# 🪙 Копилка

Персональный трекер накоплений в иностранной валюте — работает прямо в браузере, без сервера и регистрации.

[![GitHub Pages](https://img.shields.io/badge/demo-live-brightgreen?style=flat-square&logo=github)](https://YOUR_USERNAME.github.io/YOUR_REPO/)

---

## ✨ Что умеет

- **Корзины** — до 5 валютных корзин (RUB, USD, EUR, USDT, BTC и ещё ~17 валют). Откладываешь процент от каждой выплаты, фиксируешь покупку когда накопилось
- **Цели** — до 5 финансовых целей с прогресс-баром (например, «На квартиру — €30 000»)
- **Курсы валют** — загружаются автоматически, обновляются раз в сутки
- **История** — все выплаты с возможностью редактировать или удалить (с пересчётом)
- **Настройки** — процент откладывания, базовая валюта, названия корзин и целей
- **Работает офлайн** — данные хранятся в localStorage браузера

## 📱 Скриншоты

> Тёмная тема, адаптирован под мобильный, устанавливается как PWA на iOS

## 🚀 Использование

Просто открой [`index.html`](index.html) в браузере — или перейди на [GitHub Pages](https://YOUR_USERNAME.github.io/YOUR_REPO/).

При первом запуске откроются настройки: выбери валюты, процент откладывания и цели.

## 🛠 Как работает

```
Получил выплату → Ввёл сумму → X% автоматически делится по корзинам
                                        ↓
                         Когда накопилось нужное кол-во → Фиксируешь покупку
                                        ↓
                              Прогресс по целям обновляется
```

**Стек:** чистый HTML + CSS + JavaScript, без фреймворков. Курсы валют — [fawazahmed0/currency-api](https://github.com/fawazahmed0/exchange-api).
